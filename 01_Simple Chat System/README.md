# 🗨️ Simple Chat System

This project demonstrates the design and implementation of a **basic chat system** with supporting data structures and a salary tax calculator. It includes Python scripts, Jupyter notebooks, and visual diagrams to illustrate the architecture and workflow.

---

## 📂 Project Structure
- **01_simple_chat_system.py** – Core Python script for chat system logic  
- **01_simple_chat_system.ipynb** – Notebook version with step‑by‑step explanation  
- **screenshots/** – Visual diagrams of classes, data structures, and usage flow
