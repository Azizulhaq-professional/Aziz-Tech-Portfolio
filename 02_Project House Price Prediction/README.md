🏡 House Price Prediction Project

📌 Overview

This project demonstrates a complete data preparation and preprocessing pipeline for predicting house prices using Python and Pandas. The workflow transforms raw housing data into a clean, machine‑learning‑ready format, laying the foundation for model building and evaluation.

🎯 Objectives
- Prepare a structured dataset free of missing values and duplicates

- Convert categorical features into numerical format

- Build a reproducible preprocessing pipeline for ML models

📂 Repository Contents
- HousePricePrediction.csv → Raw dataset

- Project_House_Price_Prediction.ipynb → Jupyter notebook with step‑by‑step workflow

- project_house_price_prediction.py → Python script version of the pipeline

- README.md → Documentation (this file)

- /screenshots/ → Visual workflow snapshots

🛠 Tools & Libraries
- Python (>= 3.7)

- Pandas (data manipulation)

- NumPy, Matplotlib, Seaborn (optional analysis & visualization)
