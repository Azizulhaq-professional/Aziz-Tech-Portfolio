🧵 Fashion MNIST CNN – Deep Learning Project

📌 Overview

This project implements a Convolutional Neural Network (CNN) to classify images from the Fashion MNIST dataset. The dataset contains 70,000 grayscale images (28×28 pixels) across 10 fashion categories such as T‑shirts, trousers, sneakers, and bags.

The workflow covers data preprocessing, CNN model design, training, evaluation, and visualization of results.

⚙️ Tech Stack

- TensorFlow / Keras – Deep Learning framework

-  NumPy – Numerical operations

-  Matplotlib & Seaborn – Visualization

-  Scikit‑Learn – Evaluation metrics

📂 Project Structure

-  Fashion_MNIST_CNN.ipynb – Jupyter Notebook with step‑by‑step workflow

-  fashion_mnist_cnn.py – Python script for reproducible training

-  screenshots/ – Visual documentation of the workflow

- README.md – Project documentation

📊 Results

• Training Accuracy: ~88%

• Validation Accuracy: ~89%

• Test Accuracy: 88.8%

• Strong performance across most classes, with minor misclassifications in visually similar categories (e.g., shirts vs. T‑shirts).
